What It Does

Campaign Generators

Generate practical table-ready ideas for NPCs, inns, settlements, shops, monsters, encounters, treasure, magic items, dungeons, regions, and names.

Generator Packs

Shift the same generator tabs into different campaign flavors: classic open fantasy, weird fantasy, horror, low-magic grit, city intrigue, wilderness travel, naval trouble, or planar strangeness.

Campaign Vault

Save useful results into a local campaign vault, then load them later without needing an account or cloud sync.

Export and Import

Export your campaign vault as a JSON file and import it again on the same computer or another browser.

Pack-Specific Output

Results include pack lenses, pressures, faction pulls, clocks, sensory details, and table-use notes so the output feels deeper than a single random line.

Windows Download Warning

Master Generator is new and not code-signed yet, so Microsoft Defender SmartScreen may warn that it is not commonly downloaded. Only continue if you started the download from cmforgedbyfire.com or the official cmforgedbyfire GitHub release.

  1. Open the browser downloads menu next to the downloaded file.
  2. Choose Keep.
  3. If Windows shows "Make sure you trust this file", open the small arrow beside Delete.
  4. Choose Keep anyway.
  5. Extract the ZIP and open Master Generator.exe.

Notes
