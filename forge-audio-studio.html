What It Does

Procedural WAVs

Build local WAV effects with editable generator controls, waveform preview, playback, and saved JSON metadata beside each render.

Voice Lines

Generate assistant-style speech from Windows TTS or optional Edge neural voices, then shape pitch, body, clarity, grit, robot tone, and space.

Reusable Profiles

Save voice profiles as local JSON so other desktop assistants or tools can create matching response clips later.

Model-Assisted Settings

With Ollama running, a local model can convert plain-English sound ideas into generator settings for faster experimentation.

Model Requirement

Forge Audio Studio is not just a plain audio utility. The AI-assisted settings workflow requires Ollama running locally with a compatible model installed.

Recommended local models are qwen2.5-coder:7b for normal use and paul-qwen30b:latest for heavier prompt-to-settings guidance if your machine has enough room.

Manual generator controls and Windows TTS voice rendering can still run without the model helper, but the intended prompt-driven workflow needs the local model available at 127.0.0.1:11434.
