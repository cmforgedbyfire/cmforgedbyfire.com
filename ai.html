Boundary

Not the full desktop assistant

The website chat does not control private machines, read private files, use local memory, or process OCR uploads. Those stay in the local desktop build.

Public Preview

This is the first website-facing version of OTTO. Conversations may be saved to improve replies, debug issues, and review abuse. Private computer control, schedules, files, OCR, and long-term local memory are not part of this public chat.
