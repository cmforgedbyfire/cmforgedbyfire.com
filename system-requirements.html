PC Purifier

Minimum Requirements

  • OS: Windows 10 (64-bit) or newer
  • Processor: Intel Core i3 / AMD Ryzen 3 or equivalent
  • RAM: 4 GB
  • Storage: 100 MB available space
  • Display: 1280x720 resolution
  • Other: .NET 6.0 Runtime (included in installer)

Recommended

  • OS: Windows 11 (64-bit)
  • Processor: Intel Core i5 / AMD Ryzen 5 or better
  • RAM: 8 GB or more
  • Storage: 500 MB (for logs and reports)
  • Display: 1920x1080 resolution
  • Permissions: Administrator rights (for deep clean operations)

Supported Windows Versions

  • ✅ Windows 11 (all editions)
  • ✅ Windows 10 (version 1909 or newer)
  • ⚠️ Windows 8.1 (limited testing - may work but not officially supported)
  • ❌ Windows 7 and older (not supported)

FORGE

Minimum Requirements

  • OS: Windows 10 (64-bit) or newer
  • Processor: Intel Core i3 / AMD Ryzen 3 or equivalent
  • RAM: 4 GB
  • Storage: 250 MB available space
  • Display: 1280x720 resolution
  • Other: Included Python runtime inside the installer

Recommended

  • OS: Windows 11 (64-bit)
  • Processor: Intel Core i5 / AMD Ryzen 5 or better
  • RAM: 8 GB or more
  • Storage: 1 GB or more for release folders and screenshots
  • Display: 1920x1080 resolution

Platform Support

  • Windows 10/11 (primary platform)
  • macOS and Linux are not supported in this release
  • Core workflow works offline; optional AI uses the endpoint you configure

General Notes

Runtime Dependencies

All our Windows applications include their required runtimes (.NET) in the installer. You don't need to install anything separately - just run the setup and go.

Internet Connection

Not required. All our desktop applications work completely offline. Internet is only needed for downloading updates or using optional cloud features (if available).

Antivirus Compatibility

Our releases are designed to work with major antivirus software. Windows Defender SmartScreen may still warn when an installer is new or not commonly downloaded yet.

Administrator Rights

Some features (like PC Purifier's Deep Clean or system optimizations) may require administrator privileges. You'll be prompted when needed.
