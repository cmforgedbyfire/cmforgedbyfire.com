PC Purifier

Minimum Requirements

  • OS: Windows 10 (64-bit) or newer
  • Processor: Intel Core i3 / AMD Ryzen 3 or equivalent
  • RAM: 4 GB
  • Storage: 100 MB available space
  • Display: 1280x720 resolution
  • Other: .NET 6.0 Runtime (included in installer)

Recommended

  • OS: Windows 11 (64-bit)
  • Processor: Intel Core i5 / AMD Ryzen 5 or better
  • RAM: 8 GB or more
  • Storage: 500 MB (for logs and reports)
  • Display: 1920x1080 resolution
  • Permissions: Administrator rights (for deep clean operations)

Supported Windows Versions

  • ✅ Windows 11 (all editions)
  • ✅ Windows 10 (version 1909 or newer)
  • ⚠️ Windows 8.1 (limited testing - may work but not officially supported)
  • ❌ Windows 7 and older (not supported)

FORGE

Minimum Requirements

  • OS: Windows 10 (64-bit) or newer
  • Processor: Intel Core i3 / AMD Ryzen 3 or equivalent
  • RAM: 4 GB
  • Storage: 250 MB available space
  • Display: 1280x720 resolution
  • Other: Included Python runtime inside the installer

Recommended

  • OS: Windows 11 (64-bit)
  • Processor: Intel Core i5 / AMD Ryzen 5 or better
  • RAM: 8 GB or more
  • Storage: 1 GB or more for release folders and screenshots
  • Display: 1920x1080 resolution

Platform Support

  • Windows 10/11 (primary platform)
  • macOS and Linux are not supported in this release
  • Core workflow works offline; optional AI uses the endpoint you configure

  • Processor: Intel Core i3 / AMD Ryzen 3
  • RAM: 4 GB

Sky Tracker Pro

Minimum Requirements

  • OS: Windows 10 (64-bit) or newer
  • Processor: Intel Core i3 / AMD Ryzen 3 or equivalent
  • RAM: 4 GB
  • Storage: 500 MB available space after extraction
  • Display: 1280x720 resolution

Recommended

  • OS: Windows 11 (64-bit)
  • Processor: Intel Core i5 / AMD Ryzen 5 or better
  • RAM: 8 GB or more
  • Storage: 1 GB available space

Portable ZIP

Download the ZIP, extract it, then open Sky Tracker Pro.exe from the extracted folder.

Quest Genie

Status

Quest Genie is in early development. System requirements will be published closer to release. Expected to be lightweight with minimal requirements (similar to PC Purifier).

General Notes

Runtime Dependencies

Current public downloads are portable ZIPs where practical. Extract the ZIP and run the app EXE inside. Some future installer builds may include their required runtimes directly.

Internet Connection

Some tools work offline after download. Sky Tracker Pro needs internet for live weather, radar, aircraft, and space weather data. Optional cloud or AI features in other tools only use the endpoint you configure.

Antivirus Compatibility

Our releases are designed to work with major antivirus software. Windows Defender SmartScreen may still warn when a download is new or not commonly downloaded yet.

Administrator Rights

Some features (like PC Purifier's Deep Clean or system optimizations) may require administrator privileges. You'll be prompted when needed.
